How A* Algorithm Works

Heuristic Search

A* uses a heuristic function to estimate the cost from current node to goal, combining it with the actual cost from start.

Cost Calculation

f(n) = g(n) + h(n), where g(n) is cost from start, h(n) is estimated cost to goal.

Optimal Path

With an admissible heuristic, A* guarantees finding the shortest path to the destination.

Efficiency

Combines benefits of Dijkstra's algorithm and Greedy Best-First-Search for optimal performance.
